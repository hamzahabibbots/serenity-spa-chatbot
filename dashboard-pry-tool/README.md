# Car Dashboard Trim Removal Pry Tool

A 3D printable pry tool designed for safely removing car dashboard panels and trim pieces. The design resembles a nail remover with a forked tip for consistent and strong prying action.

![Concept Design](../../.gemini/antigravity/brain/a8963107-e3b7-4ed4-8277-7ce235157b04/dashboard_pry_tool_concept_1765706461585.png)

## Features

- **Ergonomic Handle**: 100mm grip section with comfortable curves
- **Forked Tip**: Like a nail remover, allows gripping and prying clips
- **Tapered Wedge**: Thin 2.5mm tip easily slides under trim
- **Strong & Durable**: Designed for PETG/ABS printing at 60-80% infill

## Files Included

1. **`dashboard_pry_tool.stl`** - Ready-to-print STL file
2. **`pry_tool.scad`** - OpenSCAD source (customizable)
3. **`generate_pry_tool.py`** - Python generator script

## Recommended Print Settings

| Setting | Value |
|---------|-------|
| Material | PETG or ABS (for strength) |
| Infill | 60-80% |
| Layer Height | 0.2 mm |
| Walls | 3-4 perimeters |
| Orientation | Flat on bed |
| Supports | Not needed |

## Usage Tips

1. Use the thin edge to slide under dashboard trim
2. Apply gentle prying pressure - the tool is strong but gentle on panels
3. The forked end is ideal for catching and releasing plastic clips
4. Work slowly around the edges to avoid scratching

## Customization

Open `pry_tool.scad` in OpenSCAD to modify:
- Handle length and ergonomics
- Head taper and width
- Fork slot size
- Overall thickness
