# Home Network Design - Cisco Packet Tracer

A comprehensive multi-LAN home network simulation with inter-LAN communication.

## Network Overview

| LAN | Name | Purpose | VLAN | Network |
|-----|------|---------|------|---------|
| 1 | Home LAN | Personal devices | VLAN 10 | 192.168.10.0/24 |
| 2 | Office LAN | Work devices | VLAN 20 | 192.168.20.0/24 |
| 3 | Guest LAN | Visitor access | VLAN 30 | 192.168.30.0/24 |
| 4 | IoT LAN | Smart devices | VLAN 40 | 192.168.40.0/24 |

## Files Included

- `network-design.md` - Complete network design documentation
- `device-configurations.md` - CLI configuration commands
- `packet-tracer-guide.md` - Step-by-step building instructions
